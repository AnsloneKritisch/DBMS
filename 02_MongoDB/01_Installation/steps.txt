HOW TO download AND SETUP mongodb
==================================


Step 1 :
--------
go to google -> type download mongodb server 
now download the msi installer
and install the complete version



Step 2 :
---------
go to google -> type download mongodb shell
and unzip in C drive



Step 3 :
---------
Add the past of server(lib) and shell(lib)  to the Environment variable



Step 4 :
---------
Then create a new folder in c drive name data and create a folder in data name db 



Step 5 :
---------
Go to CMD and Type "mongosh" and Start Wroking
